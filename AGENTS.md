# Project Overview: Income & Expense Tracker for Institutions

This project is a clean, simple, and responsive Financial Tracker web application built with **Next.js (App Router)**, **Tailwind CSS**, and **Firebase (Firestore & Auth)**. It tracks institutional incomes (dues, donations) and expenses (fixed, variable, custom), computes real-time treasury balances (Cash vs. Bank), and visualizes data using custom time-range filters (1-Month, 6-Month, 1-Year). Fully mobile responsive

---

## Technical Stack
- **Framework:** Next.js 14+ (App Router, Tailwind CSS, TypeScript)
- **Database & Auth:** Firebase Firestore & Firebase Auth
- **State Management:** React Context or simple React State (with Firestore real-time listeners `onSnapshot`)
- **Icons:** Lucide React
